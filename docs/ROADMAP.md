# Roadmap

## Q2 2024

- Public booking flow with WhatsApp confirmation.
- Order tracker with public tracking codes.
- Admin sign-in scoped to allow-listed emails.

- Store mode for first-party device sales.
- Paystack integration for repair fee top-ups.
- Pickup-comfort delivery option with agent dispatch.

- Email confirmations for store orders.
- Overstay-fee billing for unclaimed repairs.

## 2025

- Multi-station support.
- Technician dashboard with per-status SLAs.
- Bulk export of orders for accounting.
