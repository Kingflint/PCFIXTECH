# Security policy

If you discover a vulnerability, please email `Flintcheeze@yahoo.com`
instead of opening a public issue. Include reproduction steps and the
impacted area of the application.

## Scope

- Authentication bypass.
- Privilege escalation in the admin dashboard.
- Stored-credential exposure.

## Out of scope

- Self-XSS in non-admin forms.
- Issues that require physical access to a device.
